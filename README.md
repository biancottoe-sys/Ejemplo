# Casa Viva - Dashboard demo

Página web estática de ejemplo para un dashboard de casa inteligente. No controla dispositivos reales: es una maqueta visual lista para publicar en GitHub Pages.

## Archivos

- `index.html`: estructura del dashboard.
- `styles.css`: diseño responsive.
- `script.js`: pequeñas interacciones de demo.
- `assets/smart-home-hero.png`: recurso visual local.

## Publicación en GitHub Pages

1. Subí estos archivos a un repositorio.
2. En GitHub, entrá a `Settings > Pages`.
3. Elegí la rama principal y la carpeta raíz.
4. Guardá la configuración y abrí la URL que te da GitHub.
